# Golf_Rage
Sterowanie:
poruszanie strzałkami oparte na prędkości (nie można jej zmienić w powietrzu aby się zatrzymać trzeba zwolnić poprzez przyśpieszenie w przeciwnym kierunku bądź tarcie)

aby uderzyć w piłkę należy do niej podejść i nacisnąć SPACJĘ by wejść w tryb celowania
zmiana kąta strzału: strzałki lewo i prawo
zmiana mocy strzału: strzałki góra i dół
aby trzelić należy nacisnąć ponownie SPACJĘ

aby zresetować poziom należy nacisnąć guzik RESTART lub przycisk R na klawiaturze


KONTROLA DEBUGOWANIA: any przejść na następny poziom należy wcisnąć Y
